# 2. つながりタイプ

## 2-1. 欲望の核

輪の中にいたい、反応を返してほしい、必要とされたい、気持ちを共有したい。

「人が好き」という性格ではなく、関係の温度や相互反応に価値を置きやすい欲望として扱う。

## 2-2. 欲望の出し方4問＋中間確認2問

5段階。

- **TS1**(表出+)「誰かとつながりたい、関わってほしいという気持ちを、言葉や態度に出すことが多い」
- **TS2**(表出−)「寂しさや『もっと関わってほしい』という気持ちは、相手に見せず自分の中へしまう」
- **TS3**(表出+)「うれしい・寂しい・会いたいなど、関係にまつわる気持ちは相手へ伝えるほうだ」
- **TS-FIT**(適合確認)「周囲に見せるかどうかとは別に、自分の中では『人と気持ちを分かち合いたい』『必要とされたい』という感覚が強い」
- **TS-M1**(中間確認)「相手との距離や関係によって、つながりたい気持ちを見せるか隠すかを変えている」
- **TS-M2**(中間確認)「相手から反応が返ってきそうなときだけ、自分から関わりにいくことが多い」

```text
expr = TS1 + (6 - TS2) + TS3
打ち出す型: expr >= 12
内に燃やす型: expr <= 7
中間: 8..11 → TS-M1/M2
TS-FITは表出スコアに含めない
```

## 2-3. ズレ5ペア＋確認1ペア

H・Pとも「つながりたい気持ちを外へ出している度」へ統一する。

| ペア | H:本音 | P:実際の振る舞い |
|---|---|---|
| TZ1 誘い | 本音では、会いたい人には自分から声をかけたい | 実際に、会いたい人へ自分から連絡や誘いをしている |
| TZ2 反応 | 本音では、自分の話や働きかけにもっと反応してほしい | 実際に、反応がほしいことを相手へ伝えている |
| TZ3 寂しさ | 本音では、寂しいときは誰かに気づいてほしい | 実際に、寂しいときは態度や言葉で相手へ示している |
| TZ4 必要感 | 本音では、誰かの役に立ち、必要とされたい | 実際に、自分から人の輪や役割へ入っている |
| TZ5 不和 | 本音では、関係が冷えたときは早く元へ戻したい | 実際に、気まずい相手へ自分から関係修復の働きかけをしている |
| TZ6 確認 | 本音では、大切な人とはもっと気持ちを共有したい | 実際に、大切な人へ自分の気持ちを言葉で伝えている |

注意:TZ5のPは迎合ではなく、つながり欲望の表出行動。防衛カテゴリとは別に扱う。

## 2-4. 使いこなし6問＋確認2問

「実際のこの1か月」。

- **T-U-A1**「つながりたい、反応がほしいという気持ちが動いた瞬間に、自分で気づけている実感がある」
- **T-U-A2**「相手の反応を求めすぎているとき、途中でそれを自覚できることが多い」
- **T-U-R1**(逆転)「あとから振り返って初めて、『反応がほしくて動いていた』と気づくことが多い」
- **T-U-O1**「関わりにいく場面と、相手を待つ場面を、自分で選べている実感がある」
- **T-U-O2**「つながりたい気持ちを、相手へ無理をさせず関係を育てる行動へ回せている実感がある」
- **T-U-R2**(逆転)「この1か月で、反応を求めすぎたことで相手との距離が不自然になったと感じたことがある」
- **T-U-C1**「直近で、反応がほしい自分に気づいた場面を具体的に思い出せる」
- **T-U-C2**「直近で、つながりたい気持ちを関係づくりへうまく使えた場面を具体的に思い出せる」

## 2-5. 結果記述の核

- 打ち出す型:反応を返し、場を温める動きが外から見えやすい
- 内に燃やす型:関わりたい気持ちは強いが、拒絶を避けるため表には出にくい
- 抑圧:寂しさや関係への希望を隠し、平気そうに振る舞いやすい
- 演出:本音以上に明るく・親密に振る舞い、場の温度を維持しやすい
- 反転:安心できる相手には強く出し、不安な相手には急に引くなど切り替わりやすい可能性

禁止:「陽キャ」「依存体質」「愛されたいだけの人」など、性格・病理への短絡。

# 3. 読み解きタイプ

## 3-1. 欲望の核

分からないまま動きたくない、仕組みを理解したい、自分のペースと認知領域を守りたい。

単なる内向性・頭の良さではなく、理解可能性と干渉されない余白への欲望として扱う。

## 3-2. 欲望の出し方4問＋中間確認2問

- **RS1**(表出+)「分からないことや納得できない点があると、周囲へ質問や確認をすることが多い」
- **RS2**(表出−)「理解できていないことがあっても、その場では言わず一人で調べて処理する」
- **RS3**(表出+)「自分のペースや考える時間が必要なときは、周囲へはっきり伝えるほうだ」
- **RS-FIT**(適合確認)「周囲に見せるかどうかとは別に、自分の中では『きちんと理解したい』『自分のペースを守りたい』という感覚が強い」
- **RS-M1**「相手や立場によって、疑問を口にするか一人で調べるかを変えている」
- **RS-M2**「質問しても安全だと感じる場では外へ出し、評価される場では黙ることが多い」

```text
expr = RS1 + (6 - RS2) + RS3
```

## 3-3. ズレ5ペア＋確認1ペア

H・Pとも「理解や自分のペースへの要求を外へ出している度」へ統一。

| ペア | H:本音 | P:実際の振る舞い |
|---|---|---|
| RZ1 質問 | 本音では、分からない点は納得できるまで確認したい | 実際に、分からない点はその場で質問している |
| RZ2 時間 | 本音では、考える時間を十分に確保してから答えたい | 実際に、すぐ答えず考える時間が必要だと伝えている |
| RZ3 干渉 | 本音では、自分のやり方へ細かく口を出してほしくない | 実際に、任せてほしい範囲を相手へ伝えている |
| RZ4 納得 | 本音では、理由が分からない指示には従いたくない | 実際に、理由や根拠を確認してから動いている |
| RZ5 集中 | 本音では、集中しているときは人に割り込まれたくない | 実際に、集中時間や連絡可能な時間を周囲へ示している |

## 3-4. 使いこなし6問＋確認2問

- **R-U-A1**「理解できていないことへの不安や違和感が動いた瞬間に、自分で気づけている実感がある」
- **R-U-A2**「調べ続けることで行動を先送りしているとき、途中でそれを自覚できることが多い」
- **R-U-R1**(逆転)「あとから振り返って初めて、『分からないことが怖くて止まっていた』と気づくことが多い」
- **R-U-O1**「深く考える場面と、十分でなくても一度動く場面を、自分で選べている実感がある」
- **R-U-O2**「理解したい気持ちを、相手を遠ざけず質問や整理へ変えられている実感がある」
- **R-U-R2**(逆転)「この1か月で、考えすぎや説明不足によって周囲との進み方がずれたと感じたことがある」
- **R-U-C1**「直近で、分からなさへの反応に自分で気づいた場面を具体的に思い出せる」
- **R-U-C2**「直近で、理解したい気持ちを前進へうまく使えた場面を具体的に思い出せる」

## 3-5. 結果記述の核

- 打ち出す型:質問・整理・境界線を言葉にし、理解可能な状態を周囲にも求める
- 内に燃やす型:疑問や違和感を内側で処理し、外からは納得しているように見えやすい
- 抑圧:分からなさや干渉への不満を言えず、静かに消耗する
- 演出:本音以上に専門的・冷静に見せ、分からないと言いにくくなる
- 反転:得意領域では強く説明し、不確かな領域では極端に黙るなど切り替わりやすい

禁止:「頭脳派」「コミュ障」「感情がない」などの固定ラベル。

# 5. 共通スコアリング

各タイプの出し方・使いこなしは勝ち筋タイプと同じ式を使い、IDだけを差し替える。

```text
expressionScore = positive1 + (6 - negative1) + positive2
外へ出す >= 12
内へ抱える <= 7
中間 8..11 → 中間確認2問

awareness = (A1 + A2 + (6 - R1)) / 3
utilization = (O1 + O2 + (6 - R2)) / 3
>=4.0 高 / 3.0..3.9 中 / <3.0 伸びしろ
```

タイプ不適合:
- FIT<=2
- かつ baseT1-baseT2<=2 または第二適合信号との重大不整合
- H平均、使いこなし低値は参考ログのみ

ズレは各5ペアへ共通のM・方向・Cを適用し、抑圧・演出には軽度/中度/強度を付与する。
