## 1. 縦通し対象タイプの選定

| タイプ(仮) | 欲望の核 | 場面表現 | ズレ書き分け | 縦通し適性 |
|---|---|---|---|---|
| 支配系 | 勝ちたい・下に置かれたくない | ◎(会議・交渉・競争など場面が豊富) | ◎(競争心を隠す抑圧/強く見せる演出/両方の反転が全て自然に成立) | **採用** |
| 遊戯系 | つながりたい・必要とされたい | ○ | △(抑圧が「いい人疲れ」という既知の失敗パターンに寄りやすい) | 次点 |
| 観察系 | 理解したい・干渉されたくない | ○ | △(内向が既定で演出型が書きにくい) | — |
| 求道系 | 理想に近づきたい・意味が欲しい | △(抽象的で場面化に工夫が要る) | ○ | — |

**選定:支配系。** 欲望の核が最も場面化しやすく、ズレの3型(抑圧・演出・反転)がすべて現実的な人物像として成立するため、書き分けの検証効率が最も高い。防衛反応との組み合わせ(証明・分析・距離など)も多様に作れ、相性サンプルでも対照的なタイプ(遊戯系)と組ませて差を出しやすい。

## 2. 仮名称(縦通し検証用。最終名称ではない)

- 支配系 → **勝ち筋(かちすじ)タイプ**(核:勝ち・優位・取り返し)
- 遊戯系 → **つながりタイプ**(核:輪・必要とされること)
- 観察系 → **読み解きタイプ**(核:理解・自分のペース)
- 求道系 → **軸(じく)タイプ**(核:理想・意味・自分の基準)
- 出し方:外=**打ち出す型**/内=**内に燃やす型**/中間・切替=**使い分け型**

ラベル例:「勝ち筋タイプ・内に燃やす型」

## 3. 質問セット

表記:【勝】勝ち筋【つ】つながり【読】読み解き【軸】軸。選択肢順はランダム表示前提。根拠レベルは全設問「レベル1(直接測定)」。

### 3-1. 共通タイプ質問(C01〜C12・四択)

**C01** 何かを頑張るとき、一番のエネルギー源になるのは?
1. 勝負に勝つこと、結果で示すこと【勝】 2. 周りが喜んで盛り上がること【つ】 3. 分からなかったことが分かる瞬間【読】 4. 理想の形に近づいていく実感【軸】
- 測定目的:欲望の燃料源の一次判別/競合:1と4(達成文脈で近接)/識別点:他者比較の勝ちか、自己基準の接近か

**C02** 丸一日自由に使えるなら?
1. 腕試しになること(対戦、競技、ランキングのあるもの)に時間を使う【勝】 2. 気の合う人と集まって楽しむ【つ】 3. 一人で興味のあることを掘り下げる【読】 4. 自分が積み上げているもの(作品・鍛錬)に向き合う【軸】
- 測定目的:自由時間の欲望配分/競合:3と4(単独活動)/識別点:掘る対象が興味か、自己の積み上げか

**C03** グループの空気が悪くなったとき、真っ先に頭に浮かぶのは?
1. この流れをどう立て直すか、次の段取り【勝】 2. みんなが気まずくならない一言【つ】 3. なぜこうなったのか、原因の整理【読】 4. 本来どうあるべきだったか、という違和感【軸】
- 測定目的:危機時の一次関心/競合:1と3(状況対処で近接)/識別点:場を動かすか、まず理解するか

**C04** 長く続いたとき一番つらいのは?
1. 決定権がなく、言われた通りに動くだけの状態【勝】 2. 誰からも頼られず、居場所を感じられない状態【つ】 3. 自分のペースに絶えず割り込まれる状態【読】 4. 意味を感じられない作業を続ける状態【軸】
- 測定目的:本能が守るものの裏面(何を脅かされると耐えがたいか)/競合:1と3(自律の侵害)/識別点:上下関係の下位か、干渉そのものか

**C05** 高い買い物の決め手は?
1. 良いものを持っている、と胸を張れるか【勝】 2. 使う場面を想像してワクワクするか【つ】 3. 調べ尽くして納得できるか【読】 4. 自分のこだわりに合っているか【軸】
- 測定目的:消費場面での欲望/競合:3と4(吟味系)/識別点:比較の納得か、美学の一致か

**C06** 昔の自分と比べて一番「成長した」と感じるのは?
1. 勝負どころや交渉に強くなった【勝】 2. 人との輪が広がった【つ】 3. 物事を構造で捉えられるようになった【読】 4. 自分の軸がぶれなくなった【軸】
- 測定目的:自己評価の参照軸/競合:1と2(対人スキル)/識別点:優位の獲得か、輪の拡大か

**C07** 友人から相談されたとき、無意識にやりがちなのは?
1. どう動けば形勢が変わるか、打ち手を一緒に考える【勝】 2. まず気持ちに寄り添い、一緒に怒ったり笑ったりする【つ】 3. 状況を整理して選択肢を並べる【読】 4. そもそもどうありたいのかを一緒に掘る【軸】
- 測定目的:他者支援時の欲望の型/競合:1と3(解決志向)/識別点:形勢逆転か、整理そのものか

**C08** 新しい取り組みが始まるとき、最初に気になるのは?
1. 誰が決めるのか、自分は何を任されるのか【勝】 2. 誰と一緒にやるのか【つ】 3. 全体像と進め方【読】 4. その取り組みに意味があるか【軸】
- 測定目的:新環境での一次関心/競合:1と3(構造への関心)/識別点:権限の所在か、仕組みの理解か

**C09** 「もうやめたい」と感じやすいのは?
1. 頑張っても報われず、割を食っていると感じたとき【勝】 2. 一緒にやる人との関係が冷めたとき【つ】 3. やり方に細かく口を出されたとき【読】 4. やっていることの意味を見失ったとき【軸】
- 測定目的:離脱トリガー/競合:1と4(報われなさ)/識別点:損得の不均衡か、意味の喪失か

**C10** 初対面の集まりで自然にやっているのは?
1. 場の中心や流れがどこにあるかを見る【勝】 2. 話しやすい空気を作る【つ】 3. まず全体を静かに観察する【読】 4. 深く話せそうな一人を探す【軸】

**C11** 「幸せ」に一番近いイメージは?
1. 実力を認められ、対等以上に扱われている状態【勝】 2. 好きな人たちと笑い合える毎日【つ】 3. 誰にも邪魔されず興味を追える状態【読】 4. 理想とする生き方を体現できている状態【軸】
- 測定目的:到達イメージ/競合:1と4(自己実現系)/識別点:承認と序列か、生き方の一致か

**C12** 人生で一度は経験してみたいのは?
1. 大一番で結果を出して、場を沸かせる【勝】 2. 一生の思い出になる最高の瞬間を、みんなで【つ】 3. 誰も解けなかった問いを自分の力で解く【読】 4. 「あの人の生き方は本物だ」と言われる域に達する【軸】

### 3-2. 上位2タイプ比較質問(勝ち筋×他3タイプ・各2問)

同時に満たせない場面での優先を聞く(良し悪しは聞かない)。

**VS-勝つ1** 勝てるけれど場の空気が固くなるか、負けを受け入れて輪を守るか、どちらかしか選べないなら?
1. 勝ちに行く【勝】 2. 輪を守る【つ】

**VS-勝つ2** 評価がはっきり付く一人仕事と、評価は目立たないが好きな仲間との仕事、どちらかだけ選ぶなら?
1. 評価が付く一人仕事【勝】 2. 仲間との仕事【つ】

**VS-勝読1** 主導して全体を動かす役と、誰にも口を出されず自分の持ち場を極める役、どちらかなら?
1. 主導する役【勝】 2. 持ち場を極める役【読】

**VS-勝読2** 締切間際。多少粗くても先に形にして流れを取るか、納得いくまで確かめてから出すか?
1. 先に形にする【勝】 2. 確かめてから出す【読】

**VS-勝軸1** 勝ち筋はあるが自分の流儀に反するやり方と、流儀は守れるが負けが濃厚な道、どちらかなら?
1. 勝ち筋を取る【勝】 2. 流儀を守る【軸】

**VS-勝軸2** 周囲から高く評価される仕事と、自分が意味を感じる仕事、どちらか一方しか取れないなら?
1. 評価される仕事【勝】 2. 意味を感じる仕事【軸】

### 3-3. 欲望の出し方(勝ち筋タイプ用・4問+中間確認2問)

5段階(1全く当てはまらない〜5とても当てはまる)。

- **DS1**(表出+)「勝ちたい・優位に立ちたいという気持ちを、周囲に隠さず見せている」
- **DS2**(表出−)「競争心は表に出さず、結果でだけ示したい」
- **DS3**(表出+)「『負けたくない』と口に出して言うことがある」
- **DS-FIT**(タイプ適合確認・独立項目)「周囲に見せるかどうかとは別に、自分の中では『負けたくない』『優位でいたい』という感覚が強い」
  ※欲望の存在そのものを確認する。表出スコアには算入しない(§v1.0-7)
- **DS-M1**(中間確認)「相手や場面によって、競争心を見せるか隠すかを使い分けている」
- **DS-M2**(中間確認)「競争心を出すかどうかは、勝てる見込みがあるかどうかで変わる」

### 3-4. ズレ(勝ち筋タイプ用・5ペア10問+確認1ペア)

5段階。全ペアの極性は「欲望を出している度」に統一(H=本音での表出願望、P=実際の表出行動)。本音フレーム5問を出し切ってから対人フレーム5問。

| ペア | H(本音) | P(対人場面) |
|---|---|---|
| Z1 主導権 | 本音では、会議や話し合いで自分が流れを握りたい | 実際の場では、自分から流れを取りにいくことが多い |
| Z2 手柄 | 本音では、自分の貢献はきちんと自分の手柄として扱われたい | 実際には、自分の貢献を自分から伝えている |
| Z3 競争心 | 本音では、身近な相手にも負けたくないと思っている | 実際に、張り合う姿勢を相手に見せている |
| Z4 意見 | 本音では、押し切ってでも自分の案で進めたいことがある | 実際の場では、反対されても自分の案を押している |
| Z5 立場 | 本音では、軽く扱われたらその場で立場をはっきりさせたい | 実際に、軽く扱われたときはその場で示している |
| Z6 確認用 | 本音では、交渉ごとでは譲りたくない | 実際の交渉で、簡単には譲らない姿勢を見せている |

※本音質問はズレ算出専用。タイプ不適合判定にはDS-FITを用い、H平均は補助信号に留める。

### 3-5. 防衛反応(全タイプ共通・7問)

四択。選択肢は対処行動(カテゴリ:反撃CT/証明PR/距離DS/迎合AP/分析AN/自己責任化SB/感情遮断NU/固まるFR)。タイプの欲望の言い換えは禁止。

**D1** 信頼していた人が裏であなたの不利になる動きをしていたと分かった。最初に出るのは?
1. 言い返したい、問い詰めたい(CT) 2. その人から静かに離れたくなる(DS) 3. まず事実を確かめたくなる(AN) 4. 自分に原因があったのかと考え始める(SB)

**D2** その後、数日のうちに実際に取りそうな行動は?
1. 自分の行動や成果で信頼を取り戻そうとする(PR) 2. 自分の不満は抑えて、相手が納得する形へ合わせようとする(AP) 3. 接点を減らして様子を見る(DS) 4. 気持ちに蓋をして日常を続ける(NU)

**D3** 大事な場面で失敗し、大勢の前で恥をかいた。直後の頭の中は?
1. 失敗を打ち消すため、すぐ次の行動を起こしたくなる(PR) 2. 自分の恥ずかしさは抑えて、周囲が気まずくならないよう振る舞う(AP) 3. 頭が真っ白になって動けない(FR) 4. 原因の分析が勝手に始まっている(AN)

**D4** その夜、家に帰ってからのあなたは?
1. 次の機会で評価を取り戻すための準備を始める(PR) 2. 自分の希望より相手の期待に沿う方法を考える(AP) 3. 一人で反省点を整理する(AN) 4. 同じ場面を何度も反芻して自分を責める(SB)

**D5** 何をやってもうまくいかない時期が数か月続いている。起きやすいのは?
1. 小さくても結果が見える行動を増やし、状況を変えようとする(PR) 2. 自分のしんどさは後回しにして、周囲の期待へ応え続けようとする(AP) 3. 予定を減らして一人になる時間が増える(DS) 4. 感情が動かなくなり、淡々とこなすだけになる(NU)

**D6** その状態から抜けるきっかけになりやすいのは?
1. 努力の手応えや小さな成果が見え始めること(PR) 2. 誰かの希望に応える役割を引き受けること(AP) 3. 状況を整理し直して、次にできることが分かること(AN) 4. 一度すべてから離れてリセットすること(DS)

**D7** あなたが大切にしてきたものを、真っ向から「無意味だ」と否定された。反射的に出るのは?(存在意義の否定・検出1問)
1. その場で反論したくなる(CT) 2. 深く傷つくが、その場は流して距離を置く(DS) 3. 相手がなぜそう思うのか根拠を確かめたくなる(AN) 4. 「本当にそうかもしれない」と矛先が自分に向く(SB)

### 3-6. 使いこなし(勝ち筋タイプ用・6問+確認2問)

5段階。「実際のこの1か月」基準。

- **U-A1**(自覚+)「競争心や『負けたくない』が動いた瞬間に、自分で気づけている実感がある」
- **U-A2**(自覚+)「勝ち負けにこだわりすぎているとき、途中でそれを自覚できることが多い」
- **U-R1**(自覚・逆転)「あとから振り返って初めて『あのとき張り合っていたな』と気づくことが多い」
- **U-O1**(運用+)「競争心を出す場面と抑える場面を、自分で選べている実感がある」
- **U-O2**(運用+)「『負けたくない』を、人にぶつけずに行動の燃料へ回せている実感がある」
- **U-R2**(運用・逆転)「この1か月で、勝ち負けへのこだわりが人間関係をぎくしゃくさせたと感じたことがある」
- **U-C1**(確認・自覚)「直近で、競争心が動いたことに自分で気づいた場面を、具体的に思い出せる」
- **U-C2**(確認・運用)「直近で、競争心をうまく力に変えられた場面を、具体的に思い出せる」

## 3-8. 汎用フォールバックセット(低確信度ルート用)

タイプが確定しない場合に使用する。特定タイプの欲望を前提とせず、欲求・希望・不満を外へ出す過程を測る。

### 汎用出し方(GE01〜GE04・5段階)

- **GE01**(表出+)「自分が望んでいることを、周囲へ言葉で伝えることが多い」
- **GE02**(表出−)「本当に欲しいものほど、表には出さず自分の中にしまっておく」
- **GE03**(表出+)「納得していないことがあると、態度や行動に表れやすい」
- **GE04**(使い分け)「相手や場面によって、自分の希望を見せるか隠すかを意識的に変えている」

スコア:
```
genericExpr = GE01 + (6 - GE02) + GE03   # 3〜15
genericSwitch = GE04                      # 独立指標
外へ出す: genericExpr >= 12
内へ抱える: genericExpr <= 7
中間: 8..11
  GE04 >= 4 → 使い分け傾向
  GE04 <= 3 → 方向未確定。expressionConfidence=low
```

### 汎用ズレ(GZ01〜GZ05・5ペア10問)

全項目を「内側では外へ出したい度H」と「実際に外へ出している度P」に揃える。

| ペア | H:本音 | P:実際の振る舞い |
|---|---|---|
| GZ1 希望 | 本音では、自分の希望をもっと優先してほしいと思うことがある | 実際に、自分の希望を相手へ言葉で伝えている |
| GZ2 異論 | 本音では、納得できないときは違うと言いたい | 実際に、納得できないときは異論を示している |
| GZ3 境界 | 本音では、無理な頼みや負担は断りたい | 実際に、難しい依頼には断る・条件を伝える行動を取っている |
| GZ4 貢献 | 本音では、自分の努力や貢献に気づいてほしい | 実際に、自分が行ったことを必要な相手へ伝えている |
| GZ5 援助 | 本音では、困ったときは誰かに助けてほしい | 実際に、困ったときは助けを求めている |
| GZ6 確認 | 本音では、自分の都合や余裕も尊重してほしい | 実際に、予定や負担の調整を相手へ申し出ている |

GZ1〜GZ5でズレ3指標を算出し、方向不明瞭の場合のみGZ6を追加する。低確信度レポートでは「何の欲望を隠したか」を断定せず、「希望・不満・必要をどう外へ出しているか」として記述する。

### パターンCで使用する回答例

- GE01=3 / GE02=3 / GE03=3 / GE04=5 → genericExpr=9、使い分け傾向、expressionConfidence=medium
- GZ1:H2/P4(d=+2)
- GZ2:H4/P2(d=−2)
- GZ3:H3/P4(d=+1)
- GZ4:H4/P2(d=−2)
- GZ5:H3/P3(d=0)
- M=2、pos=2、neg=2 → 反転型、gapConfidence=high

## 4. スコアリング仕様(勝ち筋タイプ)

```
# タイプ(v1.0 §5に従う)
baseTypeScores[t] = C01..C12での選択数
comparisonScores[t] = 比較質問での選択数(共通12問の素点へ加算しない)
```

**相手に知っておいてもらいたいこと** — あなたが静かなのは興味がないからではなく、内側で真剣に考えているから——このタイプの組み合わせでは、この一点が共有されているかどうかで摩擦の量が変わりやすいです。[inferred]

**関係を良くするヒント** — あなたが「抑えた本音」を月に一つでも言葉にすると、場をあたためるのが得意な相手は、それを受け取る力を持っている可能性が高い組み合わせです。あなたが◯◯を我慢して譲ると、相手には「問題なし」として伝わる可能性がある——この非対称を覚えておくだけでも十分です。[inferred]

## 9. 縦通し後の自己検証

12観点で検証した結果と、発見した課題・反映内容:

**検証で問題なしと判断した項目:** 測定していない断定の混入なし(全文にタグ付与し確認)/ズレ・防衛・使いこなしはパターンBで「内に燃やす×抑圧×分析→距離×自覚高・運用低」という一人の人物像として矛盾なく接続/有料版は無料版の増量ではなくズレの方向・最大差ペア・防衛の組み合わせという有料でのみ使うデータが明確/低確信度Cは「切り替わりの観察」という固有の価値を持つ結果として成立(失敗結果に見えない)/相性は点数・優劣なし/約10分:標準39問×12〜15秒=8〜10分で妥当。

**発見した課題と対応:**

1. **ズレ小・演出軽微ユーザーにズレ固定フックが弱い**(パターンA作成中に発覚)→ v1.0仕様§12にフックのフォールバック順(ズレ→防衛→使いこなし)を追記済み。
2. **防衛7問×8カテゴリはカウントが疎**(パターンC作成中に発覚:最頻2回が3カテゴリ並立)→ v1.0仕様§6・§10に「最頻2回未満はdefenseConfidence=low」「最頻2回以上で第一防衛成立」を反映済み。カテゴリ数の削減(8→6)はリリース後の分布データを見て判断する。
3. **C05(買い物)は生活文脈で法人版に転用不可** → 法人版は別バンクという既定方針(§17)で吸収。横展開時にC05相当の職場文脈問を法人バンクに用意する。
4. **類似質問の近接**:Z3(競争心を見せている)とDS1(競争心を隠さず見せている)が近い。測定対象が異なる(ズレのP側/出し方)ため両方残すが、ユーザー体感の「同じことを聞かれた」を避けるため、出題順で最低10問以上離すことを実装要件に追加。
5. **汎用フォールバックセットが未執筆だった** → 改訂1で汎用出し方4問、汎用ズレ5ペア+確認1ペアを追加し、パターンCを実際の回答値で再計算済み。確定ルートと低確信度ルートの両方が質問から結果まで縦通し可能になった。
6. **比較質問を共通タイプ得点へ加算すると分析が汚れる** → `baseTypeScores`と`comparisonScores`を分離し、`resolvedType`で最終判定する仕様へ修正。
7. **使いこなし低値をタイプ不適合の成立条件にすると、使いこなせていない人を誤タイプ扱いする** → 使いこなし・本音低値は参考ログへ降格。
8. **防衛選択肢の一部が勝ち筋語彙へ寄っていた** → 証明を成果回復行動、迎合を自己希望を抑えて相手へ合わせる行動として中立化。
9. **演出・抑圧の強度が未区分** → M=1軽度/M=2中度/M≧3強度を追加。
