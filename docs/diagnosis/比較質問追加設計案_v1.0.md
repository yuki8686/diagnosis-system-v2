# 比較質問追加設計案 v1.0

## 0. 文書の位置づけ

本書は、上位2タイプが初回比較2問で1対1になった場合にのみ出題する、追加比較12問の確定設計である。

- 既存の比較質問12問とquestionIdは変更しない。
- 各タイプペアへ`-3`、`-4`の2問を追加する。
- 物理質問バンクは追加後163問となる。
- 一人のユーザーへ表示する比較質問は最大4問であり、診断全体の表示問数39〜47問／39〜45問は変更しない。
- 2026-07-10の条件付き承認と指定4問の文言修正を反映済みであり、本書を追加12問の実装根拠とする。
- 実装上は比較phaseを`initial`・`additional`・`completed`に分ける。`-3/-4`は`-1/-2`が1対1の場合だけ許可し、初回2対0後の追加回答、ペア外回答、重複回答、5件以上の回答を拒否する。

## 1. 設計原則

各ペアの既存2問と追加2問を合わせ、次の偏りを一方向へ固定しない。

- 個人判断／対人判断
- 攻め／守り
- 短期／長期
- 公開場面／非公開場面
- 利得フレーム／損失フレーム
- 社会的に望ましく見える選択肢

すべての設問は、同時に満たせない二つの自然な利益を対置する。片方だけを利己的、冷淡、非合理、未熟に見せない。選択肢順はセッションseedに基づきランダム化し、文中の先行提示によるアンカリングを分析ログで点検する。

行動経済学上の共通注意:

- 損失回避により、失う内容が具体的な選択肢へ回答が偏る可能性がある。
- 現状維持バイアスにより、「続ける」「保つ」選択肢が選ばれやすい可能性がある。
- 確実性効果により、結果が明確な選択肢が選ばれやすい可能性がある。
- 社会的望ましさにより、協調・慎重・信念を示す選択肢が自己申告で選ばれやすい可能性がある。
- 選択肢の利点と代償を同程度の具体性・文量で提示し、片方だけに強い否定語を置かない。

## 2. 勝ち筋 × つながり

### VS-WC-3

- **questionId:** `VS-WC-3`
- **比較タイプ:** 勝ち筋（win）× つながり（connect）
- **質問文:** 親しい相手との二人の計画で、成果を大きくできる進め方に変えるには、相手が楽しみにしていた役割を減らす必要がある。どちらを選ぶ？
- **選択肢A:** 成果が大きくなる進め方へ変え、自分が主導する
  - 対応タイプ: 勝ち筋（win）
  - 自然な利点: 結果と実現可能性を高められる
  - 代償: 相手の参加感や関係の温度が下がる可能性がある
- **選択肢B:** 成果が小さくなっても、相手が楽しみにしていた役割を残す
  - 対応タイプ: つながり（connect）
  - 自然な利点: 二人で取り組む実感と関係の温度を守れる
  - 代償: 到達できる結果を小さくする可能性がある
- **場面カテゴリ:** 非公開／対人判断／共同計画／中期
- **利得・損失フレーム:** 混合。成果拡大の利得と、参加感または成果機会の損失を対置
- **既存2問と重複しない理由:** `VS-WC-1`は公開場面で勝敗と集団の空気を比較し、`VS-WC-2`は仕事形態と評価を比較する。本問は親しい二者間で、共同参加の質と成果規模を比較する。
- **社会的望ましさの偏り点検:** Aを「相手を外す」、Bを「優しい」と表現せず、Aには成果責任、Bには共同性という正当な目的を置いた。Bだけを無条件に協調的に見せないよう、成果縮小の代償を明記した。
- **行動経済学上の懸念:** 親しい相手という設定により関係損失が過大評価される可能性がある。回答時の直近関係状態による利用可能性ヒューリスティックにも注意する。

### VS-WC-4

- **questionId:** `VS-WC-4`
- **比較タイプ:** 勝ち筋（win）× つながり（connect）
- **質問文:** 仲間と続けてきた活動で、自分に別の役割へ移る選択肢が生まれた。移れば新しい経験を得られるが、今の仲間と一緒に過ごす時間は大きく減る。どちらを選ぶ？
- **選択肢A:** 今の仲間との時間が減っても、新しい役割へ移る
  - 対応タイプ: 勝ち筋（win）
  - 自然な利点: 新しい経験と異なる役割を得られる
  - 代償: 現在の関係や共有時間を失う可能性がある
- **選択肢B:** 新しい役割へは移らず、今の仲間との活動を続ける
  - 対応タイプ: つながり（connect）
  - 自然な利点: 継続してきた関係と共有体験を守れる
  - 代償: 新しい経験や役割の変化を選ばないことになる
- **場面カテゴリ:** 非公開の個人判断／長期／所属移動
- **利得・損失フレーム:** 利得選択に伴う相互排他損失
- **既存2問と重複しない理由:** `VS-WC-2`の「評価が付く一人仕事対好きな仲間との仕事」と異なり、現在の所属から別の役割へ移る長期的な変化を扱う。仕事評価ではなく、新しい経験と所属継続の選択である。
- **社会的望ましさの偏り点検:** Aを上位・成功、Bを仲間思いと評価する語を避け、役割変更と関係継続を対等な選択として置いた。
- **行動経済学上の懸念:** 保有効果と現状維持バイアスがBを押し上げる可能性がある。一方、新奇性選好がAを押し上げる可能性がある。

## 3. 勝ち筋 × 読み解き

### VS-WR-3

- **questionId:** `VS-WR-3`
- **比較タイプ:** 勝ち筋（win）× 読み解き（analyze）
- **質問文:** 有望だが情報の少ない機会がある。今決めれば先に動けるが、調べてからでは他の人に先を越されるかもしれない。どちらを選ぶ？
- **選択肢A:** 情報が十分でなくても、先に決めて動く
  - 対応タイプ: 勝ち筋（win）
  - 自然な利点: 先行機会と主導権を得られる
  - 代償: 見落とした条件による損失を負う可能性がある
- **選択肢B:** 先を越される可能性を受け入れ、判断材料を増やしてから決める
  - 対応タイプ: 読み解き（analyze）
  - 自然な利点: 不確実性と判断ミスを減らせる
  - 代償: 機会や先行優位を失う可能性がある
- **場面カテゴリ:** 非公開／個人判断／資源投入／中長期
- **利得・損失フレーム:** 不確実な利得と、機会損失・判断損失の対置
- **既存2問と重複しない理由:** `VS-WR-2`は締切直前の完成度と速度を扱う。本問は締切のない投資的判断で、情報価値と先行優位を比較する。
- **社会的望ましさの偏り点検:** Aを無謀、Bを賢明とせず、Aには機会獲得、Bには損失抑制という合理性を与えた。
- **行動経済学上の懸念:** 曖昧性回避がBを、FOMOと希少性がAを押し上げる。回答者のリスク許容度がタイプ信号へ混入する可能性があるため、単問では解釈しない。

- **質問文:** 人前で判断を誤り、次の機会がすぐ来た。原因を十分に整理する時間はない。どちらを選ぶ？
- **選択肢A:** 不完全でもすぐ次の機会へ入り、結果を取り返しにいく
  - 対応タイプ: 勝ち筋（win）

  - 対応タイプ: 読み解き（analyze）

  - 代償: 直近の挽回機会と流れを失う可能性がある

- **行動経済学上の懸念:** 損失回避とブレークイーブン効果によりAが選ばれやすい一方、後知恵バイアスによりBが正しく見えやすい。

## 4. 勝ち筋 × 軸

### VS-WJ-3

- **questionId:** `VS-WJ-3`
- **比較タイプ:** 勝ち筋（win）× 軸（axis）
- **質問文:** より大きな決定権を持てる役割を提示された。ただし、その役割では自分の考えと完全には一致しない方針も、立場として支える必要がある。どちらを選ぶ？
- **選択肢A:** 考えと完全には一致しなくても、決定権を持てる役割を引き受ける
  - 対応タイプ: 勝ち筋（win）
  - 自然な利点: 内側から結果や方針へ影響できる立場を得られる
  - 代償: 自分の考えと役割上の判断に差が残る
- **選択肢B:** 決定権を持てなくても、自分の考えに近い立場を選ぶ
  - 対応タイプ: 軸（axis）
  - 自然な利点: 自分の基準と行動の一致を守れる
  - 代償: 結果へ介入できる範囲が小さくなる
- **場面カテゴリ:** 公開／対人・組織判断／長期／立場選択
- **利得・損失フレーム:** 権限獲得と一貫性損失、権限損失と一貫性維持
- **既存2問と重複しない理由:** `VS-WJ-1`は勝てる手段と流儀、`VS-WJ-2`は評価される仕事と意味ある仕事を比較する。本問は、決定権の範囲と自分の考えに近い立場のどちらを選ぶかを扱う。
- **社会的望ましさの偏り点検:** 方針への賛否を道徳判断にせず、「完全には一致しない」「考えに近い」という程度表現に揃えた。Aには介入範囲、Bには立場の一貫性という自然な価値を置いた。
- **行動経済学上の懸念:** 統制感の選好がAを、一貫性選好がBを押し上げる可能性がある。役割の大きさを成功と結びつけるハロー効果にも注意する。

- **選択肢A:** 目的とのずれが残っても、今ある優位と成果を活かして続ける

  - 対応タイプ: 軸（axis）

- **行動経済学上の懸念:** サンクコスト効果と保有効果がAを押し上げる。目的回復を強調すると一貫性選好がBを押し上げる可能性がある。

- **行動経済学上の懸念:** 現状維持バイアスと共有所有感がAを、新奇性選好と自己適合の強調がBを押し上げる可能性がある。

## 7. 読み解き × 軸

### VS-RJ-3

- **questionId:** `VS-RJ-3`
- **比較タイプ:** 読み解き（analyze）× 軸（axis）
- **質問文:** 人前で今後の方向を示す必要がある。確かな情報だけでは方向を一つに絞れないが、自分が望む方向ははっきりしている。どちらを選ぶ？
- **選択肢A:** 確かな情報で言える範囲だけを示し、方向の決定は保留する
  - 対応タイプ: 読み解き（analyze）
  - 自然な利点: 根拠を超えた判断を避け、後から修正しやすい
  - 代償: 方向性が曖昧なままになり、動きが遅れる
- **選択肢B:** 情報が十分でなくても、自分が望む方向を明確に示す
  - 対応タイプ: 軸（axis）
  - 自然な利点: 意味と基準に沿った方向へ人や行動を集められる
  - 代償: 新しい情報で前提が崩れた場合の修正コストが高くなる
- **場面カテゴリ:** 公開／方向提示／攻め／短中期
- **利得・損失フレーム:** 情報精度と方向形成の利得、誤方向と停滞の損失
- **既存2問と重複しない理由:** `VS-RJ-1`は自分が選択する際の根拠と信念を比較する。本問は他者へ方向を示す公開行動で、根拠範囲の限定と価値に基づく介入を比較する。
- **社会的望ましさの偏り点検:** Aを優柔不断、Bをリーダー的と見せず、Aの修正可能性とBの方向形成力を同等に記述した。
- **行動経済学上の懸念:** 曖昧性回避がA、過信と一貫性選好がBを押し上げる。公開コミットメントに対する抵抗が回答へ混入する可能性がある。

### VS-RJ-4

- **questionId:** `VS-RJ-4`
- **比較タイプ:** 読み解き（analyze）× 軸（axis）
- **質問文:** 進行中の計画に重大な問題がある可能性が出てきたが、原因はまだ特定できていない。どちらを選ぶ？
- **選択肢A:** 進行が止まっても、原因と構造が分かるまで調べる
  - 対応タイプ: 読み解き（analyze）
  - 自然な利点: 誤った修正や再発を避けやすい
  - 代償: 停滞が長引き、変える機会を逃す可能性がある
- **選択肢B:** 原因を完全に特定できなくても、望む状態へ近づく修正を先に試す
  - 対応タイプ: 軸（axis）
  - 自然な利点: 問題の可能性を放置せず、望む状態へ近づけられる
  - 代償: 原因を誤認し、別の問題を作る可能性がある
- **場面カテゴリ:** 非公開／計画修正／守り／短期
- **利得・損失フレーム:** 誤修正回避と望む状態からの乖離解消という二つの損失回避
- **既存2問と重複しない理由:** `VS-RJ-2`は時間の使い方として自由探究と理想への鍛錬を比較する。本問は問題発生後の修正順序として、原因理解と望む状態へ近づく試行を比較する。
- **社会的望ましさの偏り点検:** Aだけを慎重、Bだけを行動的と評価せず、双方に原因理解または状態改善という合理性を置いた。
- **行動経済学上の懸念:** 不作為バイアスがA、アクションバイアスがBを押し上げる。「重大な問題」が損失回避を強く喚起するため、両選択肢の損失表現を同程度に保つ。
